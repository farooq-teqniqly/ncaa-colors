NC State Wolfpack Logo Colors

The black color code for the NC State Wolfpack logo is Pantone: PMS Black 6 C, Hex Color: #000000, RGB: (0, 0, 0), CMYK: (0, 0, 0, 100). The white color code for the NC State Wolfpack logo is Hex Color: #FFFFFF, RGB: (255, 255, 255), CMYK: (0, 0, 0, 0).

NC State Wolfpack Logo

The NC State Wolfpack logo has red, black, and white colors and a big letter S object and smaller letters N and C vertically inscribed on it. The NC State Wolfpack logo meaning symbolizes the initials of the university that the team represents as well as the state bird of North Carolina, the cardinal, due to its use of the color red.
